# IPL Winner Prediction

## Overview

This project predicts the winner of IPL matches using Machine Learning techniques. Historical IPL match data is analyzed to train predictive models.

## Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Machine Learning Model Training
* Match Winner Prediction
* Model Evaluation
